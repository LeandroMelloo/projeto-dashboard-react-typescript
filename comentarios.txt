O Typescript trabalha com inferência de dados, ele identifica o tipo da varíavel, 
e não permiti inserir um novo diferente do tipo que foi atribuido inicialmente.

Interface no Typescript é definido com um contrato.

Extensões:
  - dracula, auto-rename-tag, color-highlight, material-icon-theme, svg-preview

Comando para criar um React App:
  - npx create-react-app minha-carteira --template typescript
